Mobile Automation - Appium Android setup process

Appium is an open-source tool for automating native, mobile web, and hybrid applications on iOS and Android platforms.
Appium is cross-platform: it allows you to write tests against multiple platforms (iOS, Android), using the same API.
This enables code reuse between iOS and Android testsuites.

Requirements: 1) Appium 2) Android SDK 3) Eclipse IDE 4) Selenium and Appium Java Client JAR files Android Setup: Step:1 Download Android SDK from http://developer.android.com Step:2 Go to Develop => Tools => Download => Download ADT for Mac OS X Step:3 Install the ADT to Application folder. Install all emulators with SDK 17+ Step:4 Open the ADT folder from Finder Step:5 Open Eclipse folder Step:6 Open the Eclipse.app program Step:7 Click the Window button from top bar Step:8 Click the Android SDK Managerbutton Step:9 Check all API’s 17+ and the Tools and Extras folders and install them Setup Bash-Profile: 1) Open Terminal 2) Enter $ vim .bash_profile 3) Export ANDROID_HOME = (path to SDK folder) 4) Export JAVA_HOME = SDK Path Setup Appium for Android: Check checkboxes and select a package in fron of: Package Wait for Package Launch Activity Wait for Activity Capabilities: Select Platform Name: Android Select Automation Name: Appium Select Platform Version: API 17+ Launch Device: Check the checkbox in from of Device Ready Timeout Set up 60 seconds to Device Ready Timeout If you use emulator: 1) Check the checkbox in front of Launch AVD (the emulator device should already be created) 2) In capabilities – Platform Version select the AVD selected API (Android version) If you use real device: 1) Check the checkbox in front of Device Name 2) Fill the Device Name field with device name which can be get from Android File Transfer application 3) Check the checkbox in front of Language 4) Select devices language 5) Launch the Appium app 6) Verify if all settings are corrected by taping the Doctor button 7) If all is correct tap the Inspector button to start recording the test. Appium Desired Capabilities for Android: capabilities = DesiredCapabilities.android(); capabilities.setBrowserName(BrowserType.CHROME); capabilities.setCapability("platformName", platformName); capabilities.setCapability(CapabilityType.VERSION, platformVersion); capabilities.setCapability("deviceName", deviceName); driver = new AndroidDriver(new URL("http://localhost:4723/wd/hub"), capabilities);
